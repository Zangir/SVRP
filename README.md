
# Reinforcement Learning for Solving Stochastic Vehicle Routing Problem

Here are the implementationS of our three papers on solving SVRP using RL.
The papers were done sequentially in increasing difficulty.

## RL for Solving Vehicle Routing Problem with correlated stochastic demands

Implementation of the first paper in "VRP with stochastic demands" folder.
The paper was published in IEEE Access - https://ieeexplore.ieee.org/document/10223206

## RL for Solving Vehicle Routing Problem with stochastic demands and travel costs

Implementation of the second paper in "VRP with stochastic demands and travel costs" folder.
Paper was published in ACML23 - https://arxiv.org/abs/2311.07708

## RL for Solving Vehicle Routing Problem with stochastic demands, travel costs, and time windows

Implementation of the third paper in "SVRP with time windows" folder.
Paper was submitted to AIII24 - OR workshop
